📽️ Project Vision
Abyssal Voyage is a high-performance, narrative-driven web experience created for the **Frontend Odyssey** challenge. It transforms the act of vertical scrolling into a metaphorical descent through the five biological zones of the ocean, from the sunlit surface to the crushing darkness of the Mariana Trench.

> "The sea, once it casts its spell, holds one in its net of wonder forever." — Jacques Cousteau

---

## ✨ Premium Features

### 🧊 Immersive Storytelling
Experience five distinct deep-sea layers, each meticulously crafted with unique color palettes, biological facts, and atmospheric effects:
*   **The Sunlight Zone (0-200m)** – Vibrant cyan hues, teeming with life and bioluminescence.
*   **The Twilight Zone (200-1,000m)** – The "shadow world" where light begins its beautiful fade.
*   **The Midnight Zone (1,000-4,000m)** – Absolute darkness, illuminated only by the creatures themselves.
*   **The Abyss (4,000-6,000m)** – An icy, high-pressure wasteland at the edge of the world.
*   **The Hadal Trench (6,000m+)** – The deepest scars on Earth's crust, reachable only by those who dare.

### 🎭 Animation & Interaction
*   **GSAP ScrollTrigger:** Content is revealed with precision timing and smooth easing as you descend.
*   **Layered Parallax:** Multi-depth background layers (bubbles, marine snow, sea life) create a profound sense of scale.
*   **Submerging Loader:** A professional pre-loading sequence that sets the tone for the journey.
*   **Glassmorphic UI:** Sleek, translucent interface components that react to hover and scroll states.
*   **Custom Fluid Cursor:** A reactive, water-bubble-inspired cursor that enhances the tactile feel of the site.

### 🛡️ Technical Excellence
*   **SEO Optimized:** Semantic HTML5 structure with optimized meta-tags for discovery.
*   **Fluid Responsiveness:** Built using `clamp()` and CSS Grid for a flawless experience across all devices.
*   **Performance Focused:** Lightweight components and optimized animation logic ensure a consistent 60FPS.

## 🚀 Getting Started

### Prerequisites
*   **Node.js** (v18.x or higher)
*   **npm** (comes with Node)

### Installation
1.  **Clone the Repository**
    ```bash
    git clone https://github.com/your-username/ft-odyssey.git
    cd ft-odyssey
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Launch the Voyage**
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` in your browser.

4.  **Production Build**
    ```bash
    npm run build
    ```

## 🎨 Design Philosophy
The core design concept revolves around **Visual Atmosphere**. By mapping the scroll progress to a background gradient shift—from `#79D8FF` to `#000000`—the application creates a physical feeling of sinking deeper into the ocean. 

Technical decisions were driven by the need for **Immersive Performance**. GSAP was chosen over traditional CSS animations for its superior handling of complex scroll-linked timelines and precise control over eases and durations.
